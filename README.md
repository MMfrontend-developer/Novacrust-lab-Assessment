# Novacrust-lab (Test Assessment)

This repository contains a frontend implementation for a **crypto to cash / cash to crypto / crypto loan** form interface as part of a test assessment. The project demonstrates responsive UI design with tabbed forms and custom dropdown selects using **Next.js** and **Tailwind CSS**.

---

## Features Implemented

- **Tabbed Form Interface**  
  - Crypto to Cash  
  - Cash to Crypto (Coming soon)  
  - Crypto to Fiat Loan (Coming soon)  

- **Responsive Form Layout**  
  - Form content is wider than tabs  
  - Fully responsive on mobile, tablet, and desktop  
  - Inputs and buttons styled consistently with rounded cards

- **Custom Select Components**  
  - Crypto selection dropdown  
  - Payment method selection dropdown with icons/image  
  - Dropdown width adjusted for better responsiveness

- **Navigation**  
  - Recipient details page with back arrow navigation  
  - Consistent form look across pages
